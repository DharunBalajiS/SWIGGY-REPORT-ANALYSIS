# SWIGGY-REPORT-ANALYSIS
Swiggy Report Analysis project using Power BI and Excel. Analyzed sales, orders, customer trends, and restaurant performance. Performed data cleaning in Excel and built interactive dashboards in Power BI to deliver actionable business insights.
